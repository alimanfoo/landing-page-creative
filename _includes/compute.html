Compute

Run your analysis in the cloud using these computing services.

Google Colab

Colab is an interactive computing service provided for free by Google Research and is ideal for exploratory analyses.

MalariaGEN Datalab

A JupyterHub service running in Google Cloud is available for free to observatory partners for more intensive analyses.

Terra

Terra is a cloud platform for biomedical research supporting workflows and interactive notebooks, and can be used for a wide range of analyses.

The observatory's data are hosted in Google Cloud Storage and all of the above services can be configured to run your compute with good proximity to the data.

Alternatively, data can be downloaded to your own compute resources to run analyses locally.
