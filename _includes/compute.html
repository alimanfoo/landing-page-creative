Compute

Observatory data are hosted in Google Cloud. Bring compute to the data with these cloud services.

Google Colab

Colab is an interactive computing environment provided for free by Google Cloud and useful for a wide range of analyses.

MalariaGEN Datalab

Datalab is a JupyterHub service running inside Google Cloud, available to observatory partners for more intensive analyses.

Terra

Terra supports workflows and interactive notebooks which can be run via Google Cloud.
